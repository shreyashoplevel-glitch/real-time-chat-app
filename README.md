# Real-Time Room-Based Chat App

A sleek, dark-themed real-time chat application using Express.js, Socket.io, and Vanilla JS.

## Features
- **Rooms:** Join specific chat rooms.
- **Real-Time:** Instant messaging and typing indicators.
- **Presence:** Know when users join or leave and see the online count.
- **Dark Mode:** Modern dark UI for a premium feel.

## Prerequisites
- Node.js (v14 or later)
- npm

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run the Server:**
   ```bash
   npm start
   ```

3. **Open the App:**
   Navigate to `http://localhost:3000` in your web browser.

4. **Chatting:**
   - Enter a username and a room name (e.g., "Gaming" or "General").
   - Share the room name with friends to chat together in the same room!
